Get Started

To get started import the package as follows.

  1. Open a Unity project.
  2. In the menu bar click:
    Window > Package Manager
  3. In the package window select MyAssets to find the Atlas asset store package.
  4. Confirm and import.

Initializing Atlas

  1. Create a Unity terrain.
  2. Assign "AtlasUnityTerrainRenderer" component to it.
    3. This will bind the tool with the terrain
  3. Set up Terrain Layers.
  4. Right click the hierarchy window and select:
    Atlas > Unity > Preview Volume
    5. Atlas will render your stamps in realtime within this preview volume

Editing Terrains

  1. Select the Preview Volume.
  2. Click edit to preview your stamps.
    3. The terrain will be rendered in two parts:
    • Part 1 = entire terrain at lower resolution
    • Part 2 = a smaller square at full resolution
  3. You can move the smaller preview square around to the location you want to work on, by default it will automatically center on your view.
    4. You can increase/decrease its size by adjusting the "preview render resolution" property

Placing stamps

  1. Select a stamp prefab from the project window.
  2. Drag the prefab into your scene.
  3. Position/rotate/scale the stamp to your liking.

Apply Atlas to your terrain

  1. Select the terrain.
  2. Change the heightmap resolution settings to your liking.
  3. Click render or click stop edit on the preview volume.
    atlas automatically renders when you stop editing the preview volume

Create stamps from scratch

  1. Make sure that you're in edit mode.
  2. Right click the hierarchy window and select:
    Altas > Unity > Stamp
  3. Position the stamp somewhere inside the preview volume.
  4. Select your stamp and assign a stamp asset.
  5. Your stamp will now be visible.
  6. If you want you can drag the stamp to your project panel to turn it into a prefab.

Splatmaps

  1. Make sure that you're in edit mode.
  2. Select your stamp.
  3. Click + to add a new layer.
  4. Click on preview mask.
  5. Select one of the available masks as your input.
  6. Select one of the terrain's splatmap layers as the target.
    7. The order of the targets reflects the order of the splatmaps that you configured for your terrain
  7. Change the layer properties to your liking.
    you can add more layers until you achieve the desired result

Mask painting

  1. Make sure that you're in edit mode
  2. Select your stamp.
  3. Click edit mask
  4. Tweak the brush size and opacity.
  5. Paint by clicking and dragging on the terrain within the bounds of the stamp.
  6. To invert the paint brush hold SHIFT
  7. Click apply when done.

If you're not satisfied with your results you can always.

  • Click cancel to cancel editing.
  • Click reset to revert to original mask that comes with the stamp asset.
  • Click clear to clear the mask and start from scratch.

Spline stamps

  1. Right click the hierarchy window and select:
    Atlas > Unity > Spline Stamp
  2. Assign one of the road stamp assets provided with the package.
  3. Drag a point to move it horizontally.
  4. Hold SHIFT and LEFT CLICK to add a point.
  5. Hold SHIFT and LEFT CLICK on the line to insert a point.
  6. Hold SHIFT and RIGHT CLICK on a point to remove it.
  7. Hold CTRL and drag to move a point in 3D.
  8. Hold SHIFT + CTRL and drag to scale a point.

Adding detail to your spline stamp

  1. Subdivide.
    2. adds a new node between each node
  2. Smooth height.
    3. averages all nodes height wise
  3. Smooth lateral.
    4. averages all nodes lateral wise
  4. Smooth scale.
    5. averages the scaling value of all nodes
  5. Unsubdivide.
    6. removes every other node

Aligning splines with the terrain

  1. Set heightmap opacity to 0 before dropping to floor so that the spline doesn't drop to its own height.
  2. Click drop to floor
  3. Set heightmap opacity back to 1.

Spline to Mesh

  1. Tweak the mesh properties to your likings.
  2. Click create mesh to create it.

Object Scattering

Atlas has a scattering rule workflow where each rule is an asset on its own. These rules can be part of a biome that can range from dense pine forests to desert fields or palm trees that are confined to a beach area.

The basic scattering tool is able to place trees, grass and/or prefabs. If you prefer another scattering tool Atlas will support it as long as the tool works on default Unity terrain.

Creating a scatter rule asset

  1. Create a scatterer rule asset right click the project window and select:
    Atlas > Unity > Scatter Rule Asset
  2. Select the terrain.
  3. Make sure that scattering is enabled on the "AtlasTerrainRenderer" component.
  4. Create a new element and assign the scatter rule asset you just created.

Configuring scatter rules

  1. Select the scatter rule asset.
  2. Select the splatmap layer that you want to scatter on. In this example we'll be using Splat2.
  3. Increase the min slider to finetune the threshold of asset placement.
  4. Optional: block assets from spawning by entering a tag that the scatterer should avoid.
  5. Tweak the slope and height sliders to your desire.
  6. Set spacing to 50
  7. Enable elf culling if you want objects to cull each other.
  8. Add an element to the scatter stack and feed it some big tree prefabs.
    • Set spawn radius to 25 and child count to 15
  9. Add another element with a big tree prefab.
    • Set cull radius to 2
    • Set spawn radius to 7.5 and child count to 3
  10. Add another element with a medium tree prefab.
    • Set spawn radius to 3
    • Set distance scale multiplier to 0.8
  11. Add a final element with some bush prefabs.
    • Set cull radius to 1
    • Set spawn radius to 3
    • Set distance scale multiplier to 0.5

What result will this configuration yield?

Select the terrain and click render to see the scattering tool at work.
  1. A layer of big trees will spawn with a spacing of roughly 50 meters between them.
  2. Each of these trees will spawn 15 big child trees within a 25 meter radius.
  3. These trees in turn will spawn 3 medium child trees within a 7.5 meter radius.
    4. medium trees will be scaled towards 80 % the original size based on the distance from parent
  4. The medium trees will spawn 3 bushes 3 meter radius.
    5. trees will be scaled towards half the original size based on the distance from parent

Detail layers

In this example we'll be adding grass, feel free to use this for other things like small rocks and so on.
  1. Create a scatterer rule asset right click the project window and select:
    Atlas > Unity > Scatter Rule Asset
  2. Select the splatmap layer that you want to scatter on and put it on Splat1 (most likely grass).
  3. Tweak the min and max sliders on the mask to finetune the threshold of asset placement.
  4. Tweak the slope and height sliders to your desire.
  5. Add an element to the detail stack and feed it some textures or meshes.
    6. When in HDRP you will have to assign meshes unless you have unlocked billboard grass
  6. Choose the density
  7. Click render on the AtlasTerrainRenderer component.

Multiple terrains

  1. Right click the hierarchy window and select:
    Atlas > Unity > Preview Volume
    2. Atlas will render your stamps in realtime within this preview volume
  2. Create a Unity terrain.
  3. Add 3 new terrains using the Unity + button.
  4. Select all 4 terrains and assign the "AtlasUnityTerrainRenderer" component.
  5. Resize the preview volume, position it sothat it covers your 4 terrains and enter edit mode.
  6. Place a stamp that spans the entire volume.
  7. Exit edit mode to apply the stamp to all terrians.

Creating your own stamp assets

  1. Right click the project window and select:
    Create > Atlas > Unity > Stamp Asset
  2. Name your stamp asset.
  3. Insert your textures in the correct slots.

Tips and tricks

  • Preview render resolution: use this on the preview volume to control the resolution while editing.
    • Lower values yield more performance but will result in a smaller "full resolution square"
  • Set the input of the first layer on your stamp to none.
    • This makes a good base layer and ensures that no splatmap data leaks through the stamp
  • The render order of your stamps is based on their position in the hierarchy.
  • Selecting stamps is easier through 3D gizmos.
    • If you want to select stamps in the scene view make sure they are visible and big enough
  • To easely smoother roads: subdivide twice, smooth it and unsubdivide again.
